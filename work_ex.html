Python Developer - Telegram Bot for Ayurveda-Portal

Dates: 16.05.2024 - 20.08.2024

As a Python Developer, I was responsible for developing a sophisticated Telegram bot for Ayurveda-Portal, aimed at enhancing the user experience in purchasing Ayurvedic products. The bot was designed to offer personalized wellness advice and facilitate the purchase of medicinal products. My key responsibilities included:

Setting up and configuring the bot using webhooks, FastAPI, and Ngrok to ensure smooth operation and seamless integration with external systems.

Designing and managing a PostgreSQL database, implementing schema migrations with Alembic, and creating necessary tables to support bot operations and store user data.

Developing and implementing core features such as user registration, database synchronization with the CRM system, and handling various user interactions to provide a personalized experience.

Building functionalities like a shopping cart for product purchases, a referral system for user engagement, and automated promotional messaging via Google Sheets to drive marketing efforts.

Implementing automated notifications to inform users about the status of their orders and integrating with the CRM system using aiohttp for order creation, tracking order history, and managing various keyboard interactions for improved user interface.

Creating a chat feature to facilitate communication between users and support using the aiogram library, enhancing user support and engagement.

Participating in client meetings to understand requirements, deploying the live application, and providing ongoing bug fixes and support to ensure the application's reliability and performance.
